ScrollSync

CORE_V1.0.0

System established. ScrollSync is a precision automated navigation suite designed for deep-data immersion and accessibility. Follow the protocols below to calibrate your interface.

Section 01 // Core Initialization

Global State

The Power Toggle in the header links the extension to the current tab. Without this link, all automated protocols remain offline.

Settings are Tab-Specific and persist across page loads until the tab is closed.

Execution Control

Use the Core Halt (Pause) button in the footer to temporarily suspend active movements. Essential for static data analysis.

Section 02 // Navigation Modules

Vert & Horz Glide

Linear movement along axes. The system automatically finds the primary scrollable element on the page.

  • Velocity: Adjustable 0-500 PX/S.
  • Detection: Prioritizes active scrolling elements.

Smart Pulse (Auto-Clicker)

Simulates human interaction for image galleries and readers.

  • Logic: Targets prominent img or canvas elements.
  • Position: Choose LEFT, CENTER, or RIGHT.
  • Interval: 1 to 60 seconds.

Section 03 // Haptic Interface (Gamepad)

Direct Gamepad API polling for console-like control.

Analog Mapping

Map LEFT/RIGHT sticks to control velocity based on tilt intensity.

Trigger Actions

A Button and Right Trigger are mapped to immediate Smart Click pulses.

Section 04 // Shortcut Protocol

SPACE Toggle Core Halt (Pause/Resume)
W / S Adjust Active Velocity (Inc/Dec)
A / D Manual Jump (Navigate Back/Forward)

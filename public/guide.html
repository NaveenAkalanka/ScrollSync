ScrollSync.OS // Operational Manual

CORE_V1.0.0

System established. ScrollSync is a precision automated navigation suite designed for deep-data immersion and accessibility. Follow the protocols below to calibrate your interface.

Section 01 // Core Initialization

Global State

The Power Toggle in the header links the extension to the current tab. Without this link, all automated protocols remain offline.

Settings are Tab-Specific and persist across page loads and domain jumps until the tab is closed.

Execution Control

Use the Core Halt (Pause) button in the footer to temporarily suspend all active movements. This is essential for reading stationary text or static data.

HALT_STATUS: STABLE

Section 02 // Navigation Modules

Vert & Horz Glide

Provides constant, linear movement along the specified axis. The system uses Intelligent Container Detection to automatically find the primary scrollable element on the page (even within nested frames).

  • Velocity: Adjustable from 0 to 500 PX/S.
  • Detection: Prioritizes document.scrollingElement before scanning for overflow containers.

Smart Pulse (Auto-Clicker)

Designed for image galleries, manga readers, and presentation decks. The system simulates human interaction at a fixed frequency.

  • Logic: Scans the DOM for the most prominent img or canvas element to determine the click target.
  • Positioning: Choose LEFT, CENTER, or RIGHT based on the site's navigation layout.
  • Interval: Adjustable pulse frequency (1 to 60 seconds).

Section 03 // Haptic Interface (Gamepad)

The extension supports direct Gamepad API Polling for a console-like browsing experience. Once linked, the following mappings apply:

Analog Mapping

Map the LEFT or RIGHT analog stick to control scrolling speed dynamically based on tilt intensity.

Trigger Actions

The A Button (or Bottom Face Button) and Right Trigger are mapped to trigger an immediate Smart Click pulse.

Section 04 // UI Overlay & Floating Controls

Toggle FLOAT_ON to project a persistent control interface onto the page surface. This allows for manipulation without opening the extension popup.

  • Interaction: Left-Click to drag the icon. Context-Click (Right-Click) to expand the radial menu.
  • Visual Feedback: A glowing ring indicates system status: Cyan (Active), Red (Halted), Gray (Offline).

Section 05 // Shortcut Protocol

SPACE Toggle Core Halt (Pause/Resume)
W / S Adjust Active Velocity (Inc/Dec)
A / D Manual Jump (Navigate Back/Forward)
